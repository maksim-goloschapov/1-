## Реализованный функционал

### Основы
-  **Переменные** — целочисленные, по умолчанию 0
-  **Арифметика** — сложение, вычитание, умножение, деление
-  **Скобки и приоритет операций** — поддержка сложных выражений `(5 + 3) × 2`
-  **Вывод** — отображение значений переменных в консоли

### Условия
-  **IF** — условия с операторами `>`, `<`, `==`, `!=`, `>=`, `<=`
-  **ELSE** — альтернативная ветка выполнения
-  **Вложенность** — блоки внутри IF/ELSE

### Интерфейс
-  **Drag-and-Drop** — перетаскивание блоков с автоматической нумерацией
-  **Удаление блоков** — кнопка ❌ на каждом блоке
-  **Подсветка ошибок** — красная рамка и анимация при ошибке
